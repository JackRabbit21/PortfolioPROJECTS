# Alzheimer's Disease Prediction API

A production-ready **FastAPI** backend that serves two classifiers trained on the
[AIBL (Australian Imaging, Biomarker & Lifestyle)](https://aibl.csiro.au/) dataset
to predict Alzheimer's Disease at baseline from tabular clinical features.

**Stack:** Python 3.11 · FastAPI · SQLite · SQLAlchemy · scikit-learn · Docker

---

## Model Performance

| Model              | Accuracy | ROC-AUC |
|--------------------|----------|---------|
| MLP Neural Net     | 94.2 %   | 0.975   |
| Logistic Regression| 94.8 %   | 0.977   |

## Quick Start

### Option A — Docker (recommended)

```bash
# Build and run
docker-compose up --build

# The API is now available at http://localhost:8000
```

### Option B — Local Python

```bash
# Install dependencies
pip install -r requirements.txt

# Train models (point to your copy of aibl_merged_clean.csv)
python train.py --data ./Data/aibl_merged_clean.csv

# Start the server
uvicorn app.main:app --reload
```

## API Endpoints

All endpoints except `/health` require **HTTP Basic Auth** (default: `admin` / `secret`).

| Method | Endpoint                | Description                              |
|--------|-------------------------|------------------------------------------|
| GET    | `/health`               | Service status + model metrics (no auth) |
| POST   | `/predict`              | Submit patient features → AD prediction  |
| GET    | `/predictions`          | List stored predictions (paginated)      |
| GET    | `/predictions/{id}`     | Retrieve a single prediction             |
| DELETE | `/predictions/{id}`     | Delete a prediction record               |
| GET    | `/docs`                 | Interactive Swagger UI                   |

## Example Usage

### POST `/predict`

```bash
curl -u admin:secret -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "features": {
      "PTGENDER": 2,
      "APGEN1": 4,
      "APGEN2": 4,
      "CDGLOBAL": 2.0,
      "LIMMTOTAL": 4,
      "LDELTOTAL": 2
    },
    "model": "mlp"
  }'
```

**Response:**

```json
{
  "id": 1,
  "model": "mlp",
  "prediction": 1,
  "probability": 0.9716,
  "label": "Alzheimer's Disease",
  "created_at": "2026-03-19T11:03:52.586067"
}
```

All 20 feature fields are **optional** — any omitted field is imputed with the
training-set median. The `model` field defaults to `"mlp"`.

### GET `/predictions`

```bash
curl -u admin:secret "http://localhost:8000/predictions?limit=10&model_name=mlp"
```

## Features

The 20 clinical/biomarker features (all optional at inference time):

| Feature    | Description                          |
|------------|--------------------------------------|
| PTGENDER   | Patient gender (1=Male, 2=Female)    |
| APGEN1/2   | APOE alleles (2, 3, or 4)            |
| CDGLOBAL   | CDR global score (0=normal → 3=severe)|
| LIMMTOTAL  | Logical Memory Immediate Recall      |
| LDELTOTAL  | Logical Memory Delayed Recall        |
| AXT117     | TSH (thyroid stimulating hormone)    |
| BAT126     | Vitamin B12                          |
| HMT3–102   | Haematology panel (6 values)         |
| RCT6–392   | Clinical chemistry panel (4 values)  |

## Configuration

Override defaults via environment variables (or a `.env` file with Docker):

| Variable       | Default                        | Description             |
|----------------|--------------------------------|-------------------------|
| `API_USERNAME` | `admin`                        | Basic auth username     |
| `API_PASSWORD` | `secret`                       | Basic auth password     |
| `DATABASE_URL` | `sqlite:///./alzheimer_api.db` | SQLAlchemy database URL |
| `ARTIFACTS_DIR`| `./artifacts`                  | Path to model artifacts |
